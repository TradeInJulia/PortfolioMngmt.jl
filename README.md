# PortfolioMngmt.jl

**PortfolioMngmt.jl** is a [Julia](http://julialang.org) package for portfolio management. Portfolio.jl can help with capital allocation of risky assets and efficient diversification methods. The package is based on the knowledge from [MicroMasters Program in Finance](https://micromasters.mit.edu/fin/) on [Edx](https://courses.edx.org/) platform. This will also be used for portfolio management for package [Varthaka](https://github.com/TradeInJulia/Varthaka.jl).
